2016 workflow notes

Folders:
/builds		= final, compiled files
/components	= various files used to build project, ex: SASS

To setup Git using GitHub:
got to desired dir
$ git init
$ git add .

Commit to Master:
in Desktop, Commit
- or -
$ git commit -m “Comment”

In Desktop, Publish (can make private).


Install Gulp and other components to project, as devDependency:
npm install gulp --save-dev
npm install gulp-ruby-sass --save-dev
npm install gulp-util --save-dev
npm install gulp-concat --save-dev 
npm install jquery --save-dev
npm install gulp-livereload --save-dev
npm install gulp-if --save-dev 
npm install gulp-autoprefixer --save-dev 
npm install gulp-postcss --save-dev 
npm install gulp-sourcemaps --save-dev 
npm install gulp-uglify --save-dev 

in _components/SASS:
$ bourbon install
$ neat install
$ bitters install

——— Troubleshooting ———
This will delete the Bourbon files from the current directory and regenerate it:
$ bourbon update

Update Ruby gems:
$ sudo gem update

Install/Uninstall gems:
$ sudo gem un/install ###

To update npm:
npm update {to update local nom packages; dir where package.json is}
sudo npm install npm -g {to update global npm packages}

To create a package.json file:
$ npm init
