2016 workflow notes

update Ruby gems:
$ sudo gem update

Install/Uninstall gems:
$ sudo gem un/install ###

To create a package.json file:
$ npm init

Folders:
/builds		= final, compiled files
/components	= various files used to build project, ex: SASS

To setup Git:
got to desired dir
$ git init

$ git add .

$ git commit -m “Comment”

in project dir:
$ compass create <myproject> --bare

Install Gulp and other components to project, as devDependency:
$ npm install --save-dev gulp
$ npm install --save-dev gulp-util
$ npm install --save-dev gulp-concat
$ npm install --save-dev gulp-browserify
$ npm install --save-dev jquery
$ npm install --save-dev gulp-livereload
$ npm install --save-dev gulp-sass
/* $ npm install --save-dev gulp-compass */
$ npm install --save-dev gulp-if
$ npm install --save-dev gulp-autoprefixer
$ npm install --save-dev gulp-postcss
$ npm install --save-dev gulp-sourcemaps
$ npm install --save-dev gulp-uglify



in _components/SASS:
$ bourbon install
$ neat install
$ bitters install


$ npm install --save-dev gulp-coffee
$ npm install --save-dev mustache


This will delete the Bourbon files from the current directory and regenerate it:
$ bourbon update
